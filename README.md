### 1. 동기 (Motivation)

<br/>

🥅 &nbsp; **_목적_** :   

* 라이브러리 없는 개발을 통해 native 개발 환경을 이해하기 

* 주어진 기능을 최대한 빨리 구현할 수 있는 미니멀한 개발 환경 찾기 

* 현재의 요구사항의 구현에 적합한 모듈화 수준 찾기 (모듈화 vs 하드코딩)
